## Projetos em destaque

### Sistema de Clientes e Atendimentos

Aplicação web full stack para gerenciamento de clientes e acompanhamento de atendimentos.

**Principais funcionalidades:**

- Autenticação de usuários
- Cadastro, edição e exclusão de clientes
- Registro e acompanhamento de atendimentos
- Filtros e validação de dados
- Rotas protegidas
- Dashboard de indicadores
- Práticas básicas de segurança

**Tecnologias:** JavaScript, Node.js, HTML5, CSS3 e persistência em JSON.

### Dashboard de Vendas e Estoque

Dashboard desenvolvido para análise de indicadores comerciais e acompanhamento de estoque.

**Indicadores apresentados:**

- Faturamento total
- Ticket médio
- Quantidade de vendas
- Produtos mais vendidos
- Produtos com estoque baixo

**Tecnologias:** JavaScript, HTML5 e CSS3.
